파주 버스 관측 예측 앱

로컬 실행 방법
1. 앱_실행.bat 파일을 더블클릭합니다.
2. 브라우저가 열리면 http://127.0.0.1:8765 에서 앱을 사용합니다.
3. 휴대폰은 PC와 같은 Wi-Fi에 연결한 뒤, 실행 창에 표시되는 IPv4 주소로 접속합니다.
   예: http://192.168.0.12:8765
4. 휴대폰 브라우저 메뉴에서 홈 화면에 추가를 누르면 앱처럼 실행할 수 있습니다.
5. 창을 닫으면 앱 서버도 종료됩니다.

주요 기능
- 정류소명 또는 번호 검색
- 선택 정류소의 경유 노선 조회
- 도착정보 조회 및 보정 예상시간 표시
- 노선 형상과 정류소 지도 표시
- 노선 지도에 첫 번째/두 번째 도착 예정 버스 위치 추정 표시
- 실제 도착 버튼으로 예측 오차 기록
- 같은 정류소/노선의 평균 오차를 다음 예상시간에 반영
- 평균 구간 이동시간과 정차시간을 조정하는 파주 이동시간 시뮬레이션
- 휴대폰 홈 화면 추가용 앱 아이콘과 설치 설정
- Android/iPhone 홈 화면에서 잘 보이도록 SVG와 PNG 아이콘 포함

참고
- 로컬 API 키는 config.env 파일에 들어 있습니다.
- 외부 배포에서는 BUS_API_KEY 환경변수에 API 키를 넣습니다.
- 관측 기록은 현재 브라우저의 로컬 저장소에 저장됩니다.
- 휴대폰에서 접속이 안 되면 Windows 방화벽에서 Python의 개인 네트워크 접속을 허용해야 할 수 있습니다.

외부 공개 배포
- Render 배포 방법은 DEPLOY_RENDER.txt 파일을 보세요.
- Netlify 배포 방법은 DEPLOY_NETLIFY.txt 파일을 보세요.
- 배포 후에는 PC 없이 휴대폰 단독으로 공개 URL에 접속해서 사용할 수 있습니다.
