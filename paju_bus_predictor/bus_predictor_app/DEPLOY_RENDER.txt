Render로 외부 공개 웹앱 만들기

결과
- 배포가 끝나면 https://paju-bus-predictor.onrender.com 같은 공개 주소가 생깁니다.
- 휴대폰은 PC 없이 그 주소로 바로 접속합니다.
- 브라우저 메뉴에서 홈 화면에 추가하면 앱처럼 실행할 수 있습니다.

1. GitHub에 올리기
- bus_predictor_app 폴더 전체를 GitHub 저장소에 올립니다.
- config.env 파일은 올리지 마세요. .gitignore에 제외되어 있습니다.

2. Render에서 Web Service 만들기
- https://render.com 에 로그인합니다.
- New + 메뉴에서 Web Service를 선택합니다.
- GitHub 저장소를 연결합니다.
- Root Directory가 필요하면 bus_predictor_app 폴더를 지정합니다.

3. 설정값
- Environment: Python
- Build Command: pip install -r requirements.txt
- Start Command: python server.py

4. 환경변수 등록
- Environment 탭에서 BUS_API_KEY를 추가합니다.
- 값에는 공공데이터 API 키를 넣습니다.
- 이 값은 코드나 휴대폰 화면에 노출되지 않습니다.

5. 배포 후 사용
- Render가 만든 공개 URL을 휴대폰에서 엽니다.
- Android Chrome: 메뉴 > 홈 화면에 추가
- iPhone Safari: 공유 버튼 > 홈 화면에 추가

주의
- Render 무료 플랜은 한동안 접속이 없으면 잠들 수 있어 첫 접속이 느릴 수 있습니다.
- 공공 API 일일 호출 제한이 있으므로 새로고침을 너무 자주 하면 제한에 걸릴 수 있습니다.
